# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Initial release of ActiveDataFlow core gem
- Source base class for data sources
- Sink base class for data destinations
- DataFlow module for orchestration
- Message types: Unconstrained, CloudEvent, CloudEventLd
- Registry system for plugin management
- Configuration validation with type checking
- Structured logging interface
- Version compatibility checking
- Test helpers for testing DataFlows
- Comprehensive documentation and examples
